# 📱 todo-pwa

> Progressive Web app built with `create-react-app`

## 🚀 Getting Started

- `$ git clone https://github.com/ragmha/todo-pwa.git`

- `$ yarn`

- `$ yarn start`

## 🐼 Tests

- `$ yarn test`

## 👷‍♂️ Build

- `$ yarn build`
